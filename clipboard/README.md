# NodeBot - Clipboard

NodeBot allows you to set or retrive clipboard data

## Install
Run the following command in your Node-RED user directory 
```
npm install nbr-clipboard
```

If you didnt install ws before, you must install ws also. 
```
npm install ws
```

## Usage
